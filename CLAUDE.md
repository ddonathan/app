# CLAUDE.md — Task App

## Stack
- **Frontend:** React + Vite + TypeScript
- **Backend:** Convex (schema in `convex/schema.ts`, functions in `convex/tasks.ts`)
- **API:** HTTP actions in `convex/http.ts` (bearer token auth)
- **Hosting:** Vercel (frontend) + Convex (backend)

## Code Quality
- **Biome** for linting and formatting — run `npx biome check --write .` before committing
- Do NOT modify `convex/_generated/` — these are auto-generated by Convex
- TypeScript strict mode in `src/`, Convex handles its own type checking

## Build
- `npm run build` → `vite build` (no tsc — Convex types are checked during deploy)
- `npm run dev` → local dev server
- Deploy: `npx convex deploy --cmd 'npm run build'`

## Conventions
- Dark theme: bg #0d1117, surface #161b22, border #30363d, text #e6edf3, accent #58a6ff
- Mobile-first responsive design
- Tag prefixes: @ (context), p- (person), c- (client), # (project), ! (priority)
